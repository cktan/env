# Note on setting up a new sprite VM

. gh auth login
. gh repo clone env
. cd env
. bash sprite-init.sh


## Setting up emacs

. em ~/.emacs   # ignore errors
. M-x list-packages
. select go-mode and install it
. select elixir-mode and install it
. C-x C-c

. em 
. M-x menu-bar-open
. Select Options -> Theme -> wombat
. Select [Save Theme]
. C-x C-c
